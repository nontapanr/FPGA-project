## Digital Locker Project (with FPGA)

### ที่มาของงาน
ในสถานที่ทำงาน หรือสถานศึกษาล้วนทั้งเป็นสถานที่ที่มีคนเยอะ และเป็นสถานที่สาธารณะ ดังนั้นแล้วเมื่อเราต้องการทำธุระในสถานที่แบบบนี้ เราต้องการพื้นที่ไม่มากก็น้อย ที่จะเก็บสิ่งของมีค่าของเราเอาไว้อย่างปลอดภัย มีการใช้งานที่เข้าใจง่าย และดูร่วมสมัยไปในคราวเดียวกัน จึงทำให้เกิดเป็นนวัตกรรม Digital Locker ขึ้นเพื่อตอบโจทย์ที่เก็บของที่ได้กล่าวมาข้างต้น

### วัตถุประสงค์
1. เพื่อความสะดวกสบายในการใช้งานที่เก็บของ
2. เพิ่มความปลอดภัยที่เกิดจากการพยายามโจรกรรม โดยใช้ช่องโว่วของระบบที่เก็บของแบบธรรมดา
3. เพิ่มความสามารถพิเศษที่ตอบโจทย์การใช้ชีวิตในสมัยนี้

### โมดูล
- Numpad ที่ต่อเข้ากับ Arduino เพื่อรับค่า Data (รหัสผ่าน) โดยเมื่อเรากดปุ่มรหัส 4 ตัว Arduino จะทำการประมวณผล และส่งข้อมูลออกไปเป็น Binary โดยจะสั่งให้ค่าส่ง High, Low แทน 0,1 และจะส่งข้อมูลแบบ Parallel in Parallel out และนำไปเก็บ ใน Register
- 7SEGMENT แสดงผลรหัสผ่านที่รับเข้ามาจากทาง Arduino ซึ่งอยู่ใน Register โดยจะผ่านการแปลงข้อมูลจาก BCD ไปเป็น 7SEGMENT ก่อน 
- Locker โดยตัว Locker นี้จะทำงานเมื่อเราสั่ง Logic High โดยจะมีลิ้นโลหะขึ้นมากั้น โดยการตรวจสอบ เราจะตรวจสอบความถูกต้องของรหัส โดยการนำข้อมูลที่จะถูกส่งไปแสดงผลที่ 7SEGMENT มาทำการ Comparator กับรหัสที่ถูกต้อง เมื่อรหัสถูกต้องจะทำการส่ง Logic Low ไปยังตัว Locker
- Emergency Buzzer โดยโมดูลนี้ทำหน้าที่เตือนภัยเมื่อรหัสที่ถูกป้อน เข้ามานั้นผิดเกินกำหนด โดยจะส่งสัญญานโดยใช้ Buzzer ในการเตือน โดยจะมีขา TC ต่อออกมาจากตัว Comparator และส่งต่อไปยัง Counter คอยนับเมื่อถึงที่ตั้งค่าไว้ จะส่ง Logic High ไปยัง Buzzer และจะคอยมี LED แสดงผลค่ารหัสที่เข้าไปถูกหรือผิด
